# JokeCraft Network

**A custom Minecraft server experience built on Minestom**

---

## Repositories

### Core Projects

- **[JokeLang](https://github.com/JokeCraftMC/JokeLang)** *Public*  
  Multi-language support system with community-driven translations. Contributions welcome!


- **[JokeStom](https://github.com/JokeCraftMC/JokeStom)** *Private*  
  The core Minestom-based Minecraft server implementation with modular architecture, custom gameplay systems, and advanced player management.



- **[JokePack](https://github.com/JokeCraftMC/JokePack)** *Private*  
  Official custom resource pack featuring unique textures, sounds, and visual enhancements. Automatically builds and distributes new versions via GitHub Actions.



- **[JokeWeb](https://github.com/JokeCraftMC/JokeWeb)** *Private*  
  Official website and web portal providing server information, live updates, and community features.

---

## ✨ Features

- **Minestom-Powered**: Built on the lightweight, high-performance Minestom server framework
- **Custom Resource Pack**: Unique textures and sounds with automatic updates
- **Multi-Language**: Support for multiple languages with community translations

---

## 🛠️ Tech Stack

- **Server**: [Minestom](https://minestom.net/) (Minecraft 1.21+)
- **Language**: Java 25
- **Build**: Gradle
- **Database**: MySQL/MariaDB
- **CI/CD**: GitHub Actions
- **Hosting**: Self-hosted / Pterodactyl
